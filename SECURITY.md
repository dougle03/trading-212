# Security

Use a read-only Trading 212 API token only.

This integration must not trade, cancel orders, edit pies, deposit, withdraw, or otherwise mutate a Trading 212 account.

Do not share API keys, API secrets, account identifiers, balances, positions, logs, screenshots, or diagnostics containing private account data.

If a token or secret may have been exposed, revoke it in Trading 212 immediately and create a new read-only token.
